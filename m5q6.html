Differences Between HTML and HTML5

HTML HTML5
HTML does not support audio and video elements natively. HTML5 introduces native support for audio and video with the <audio > and <video > elements.
HTML requires the use of third-party plugins (like Flash) for multimedia content. HTML5 allows multimedia content to be embedded directly without the need for plugins.
HTML has limited form input types (like text, password, checkbox). HTML5 introduces new input types (like email, date, range) and attributes for better form handling.
